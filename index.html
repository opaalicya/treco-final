Sobre meu site Salon-line

0 projeto é uma página web criada com HTML & CSS, destinada a exibir nossas Luminarias. O objetivo é criar um ambiente virtual onde seja possível comprar nossas luminarias, fornecendo uma descrição breve de cada um, incluindo para que tipo de arbientes o e uma opção de compra.

A estrutura do site é baseada em HTML, que define a semántica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia disposição dos elementos.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (container) que spresenta a propósito do site e a coleção dos produtos em uma seção flexível (.Produto). Cada produto é destacado em seu próprio contêiner (.Produto), mostrando uma imagem da capa, detalhes do cremes e um link de compra. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de cores suaves e botões interativos. O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária. A escolha da tipografia e do esquema de cores contribui para a atmosfera acolhedora do site, incentivando a exploração das luminarias.

Minha Livraria

O projeto é uma aplicação educativa interativa focada em ensinar conceitos básicos de literatura. Utiliza uma série de scripts associados a diferente scripts e cenários para criar uma experiência de aprendizado envolvente. O objetivo é familiarizar os leitores.

Cada leitor no projeto possui scripts que definem seu comportamento em cada Manga/Comic específicos. Por exemplo, ao clicar em determinados mangas/comics, os leitores podem "ter" uma experiencia, o que leva a uma mudança de cenário e a exibição de informações relevantes ou a visualização de uma imaginação específica.

sobre os cards

O projeto consiste em uma aplicação web interativa, desenvolvida com HTML, CSS, e JavaScript, que simula uma experiência narrativa centrada no tema de estilos de jogos. O objetivo é engajar os jogadores em um cenário fictício em diferentes estilos.

O HTML estrutura o conteúdo da página, dividindo-o em seções como a tela inicial, caixa de perguntas, alternativas de resposta, e o resultado final baseado nas escolhas do jogador. O CSS é utilizado para estilizar a página, empregando um esquema de cores futurista e layout responsivo para melhorar a experiência do usuário. JavaScript é usado para adicionar interatividade ao jogo, manipulando DOM para atualizar o conteúdo da página conforme as escolhas do jogador, e determinando o fluxo da narrativa através de uma série de perguntas e respostas.

Os scripts JavaScript incluem módulos para gerar nomes aleatórios, definir as perguntas e as lógicas de jogo, como iniciar o jogo, apresentar perguntas e alternativas, processar as escolhas dos jogadores, e exibir o resultado final. Este design modular facilita a manutenção e a expansão futura do jogo. A importância das ferramentas utilizadas (HTML, CSS, JavaScript) reside na criação de uma experiência interativa que não apenas entretém, mas também educa os jogadores sobre a complexidade ética e social, incentivando a reflexão crítica sobre o papel da tecnologia em nossas vidas.
